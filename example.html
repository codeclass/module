Module - is easy and fast JQuery plugin, needed to structure the files in the project, to separate logic from presentation. Convenient for code reuse and team development. Simply create your own modules and use them to create powerful client-side applications

Simple module "hello world" in path "hello/main" run with defaults


    $('#example1').module('hello/main', {});


Simple module "hello world", run with custom parameters


    $('#example2').module('hello/main', {
        text: 'hello world example',
        caption: 'Hello world!'
    });

This example shows, how to access module methods, and how to use callbacks


    $('#example3').module('hello/main',{
        onTextAdd: function(){
            alert('somebody add text?');
        }
    });

    setTimeout(function(){
        $('#example3').module('hello/main').addText('. Added text.');
    }, 5000);

This example shows, how to integrate one module into another


    $('#example4').module('hello/main', {
        onInit: function(obj){
            obj.setText($('<a></a>').module('tag/a', {
                template: 'link. Click me!',
                onClick: function(){
                    alert('Ok, you\'ve clicked me');
                }
            }));
        }
    });
